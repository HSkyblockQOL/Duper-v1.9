# Duper-v1.9

// Mod installation (forge 1.18.9 required)

Download from link

Go to %appdata% in explorer

Go to your .minecraft folder

Drag mod into your forge 1.18.9 "mods" folder

Open minecraft, and open Skyblock, Enjoy!
